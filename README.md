# Transfer Student ChatBot
 This application serves as a tool for students who are attending a Community College and are intereested in transferring to a four-year university. The chat bot answers questions concerning GPAs, deadlines, application process, competitiveness, and school specific queries. 

Type your question on the input or select one of the prelisted tags and chat bot will reply. 

To run this project, open a terminal in the project folder and start a local HTTP server by running: `npx serve . -l 8000`
